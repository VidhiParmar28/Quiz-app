# Quiz App ❓

A multiple-choice quiz application built using HTML, CSS, and JavaScript.

## Features
- Multiple questions
- Dynamic options
- Progress indicator
- Next button navigation
- Quiz completion message

## Technologies Used
- HTML
- CSS
- JavaScript

## Concepts Used
- Arrays
- Objects
- map()
- Functions
- DOM Manipulation
- Event Handling

## What I Learned

- Using arrays of objects
- Creating reusable functions
- Updating the UI dynamically
